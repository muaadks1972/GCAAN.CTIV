# GCAAN - نظام إدارة أنشطة الموظفين

## نظرة عامة
تطبيق جوال (Android/iOS + Web) باللغة العربية للشركة العامة لإدارة المطارات والملاحة الجوية (GCAAN). يوفر تدفق موافقات هرمي متعدد الأدوار مع تقارير شاملة ومؤشرات أداء (KPIs).

## هيكل الأدوار
1. **المدير العام** — يُنشئ مديري الأقسام + يعتمد نهائياً + يدير الأقسام
2. **مدير القسم** — يُنشئ مديري الشعب + الشعب + يوافق/يعدل النشاطات ويرفعها للمدير العام
3. **مدير الشعبة** — يُنشئ الموظفين + يوافق على النشاطات ويرفعها لمدير القسم
4. **الموظف** — يُدخل نشاطاته (تاريخ، نوع، القسم المستهدف، ملاحظات)

## سير العمل (Workflow)
`الموظف يُنشئ نشاط → pending_division → مدير الشعبة يوافق → pending_department → مدير القسم يوافق/يعدل → pending_gm → المدير العام يعتمد → approved`

## الميزات
- تسجيل دخول JWT مع صلاحيات مبنية على الأدوار
- شاشة إدارة الأقسام للمدير العام (إنشاء أقسام جديدة)
- شاشات إنشاء المستخدمين المتدرجة
- بطاقات KPI في الرئيسية (إجمالي، معتمد، بانتظار، نسبة الاعتماد)
- تقارير: أسبوعي، شهري، حسب الأقسام، نسب إنجاز الموظفين
- شاشة مؤشرات الأداء (KPI) مع تصنيف الموظفين (ممتاز/جيد جداً/جيد/يحتاج تحسين)
- صيغة KPI: 50% نسبة الاعتماد + 30% حجم النشاط + 20% سرعة الاعتماد

## التقنيات
- **Backend:** FastAPI + Motor (MongoDB) + JWT (python-jose) + bcrypt
- **Frontend:** Expo Router (React Native) — SDK 54 — RTL Arabic
- **Design:** Navy/Sky-blue palette, Cairo/Tajawal fonts, GCAAN logo

## بيانات الدخول الافتراضية
- المدير العام: `admin / admin123`

## الأقسام الافتراضية (تم seed تلقائي)
1. قسم العمليات الجوية
2. قسم الملاحة الجوية
3. قسم الأمن والسلامة
4. قسم الصيانة والهندسة
5. قسم الشؤون الإدارية والمالية

## API endpoints الرئيسية
- `POST /api/auth/login`, `GET /api/auth/me`
- `GET/POST /api/departments`, `GET/POST /api/divisions`
- `GET/POST /api/users`
- `GET/POST /api/activities`, `POST /api/activities/{id}/action`
- `GET /api/reports/{summary|weekly|monthly|by-department|completion-rates|kpis}`
